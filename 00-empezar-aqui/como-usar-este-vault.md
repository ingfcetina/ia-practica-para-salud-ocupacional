# Cómo usar este vault

tags: #obsidian #guia

Este repositorio está escrito como vault de Obsidian. Las notas se conectan con enlaces internos como [[00-empezar-aqui/mapa-del-vault|Mapa del vault]].

## En Obsidian

1. Descarga o clona el repositorio.
2. Sigue [[00-empezar-aqui/instalacion-y-preparacion|Instalación y preparación]] si todavía no tienes Obsidian o Python.
3. Abre la carpeta como vault.
4. Empieza por [[00-empezar-aqui/mapa-del-vault|Mapa del vault]].
5. Usa las secciones `Relacionado` para avanzar.

## En GitHub

También puedes leer los Markdown directamente. Los wikilinks sirven como convención de navegación aunque GitHub no siempre los convierta en enlaces clicables.

## Convenciones

- Notas hub/MOC para cada sección.
- Wikilinks internos, sin rutas absolutas locales.
- Tags para búsqueda.
- Contenido breve, progresivo y práctico.
