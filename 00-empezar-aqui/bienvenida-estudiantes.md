# Bienvenida para estudiantes

tags: #inicio #estudiantes #salud-ocupacional

Este vault es una guía práctica para empezar a programar con ayuda de inteligencia artificial en temas de salud ocupacional.

No tienes que saber programar antes de comenzar. El objetivo es aprender una forma de trabajo:

```text
Problema profesional → datos seguros → pregunta clara → prototipo pequeño → revisión humana → mejora
```

## Qué vas a practicar

- Pensar problemas de salud ocupacional como tareas que se pueden analizar.
- Usar IA para pedir explicaciones, ejemplos y borradores de código.
- Leer datos sintéticos con Python.
- Crear gráficos simples.
- Probar una página web interactiva sencilla.
- Reconocer límites éticos y de privacidad.

## Qué no busca este material

- No busca formar personas expertas en programación en una hora.
- No reemplaza formación clínica, legal, técnica ni normativa.
- No autoriza usar datos reales en herramientas públicas de IA.
- No convierte respuestas de IA en decisiones profesionales automáticas.
