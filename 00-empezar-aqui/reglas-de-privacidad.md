# Reglas de privacidad

tags: #privacidad #etica #salud-ocupacional

Antes de usar IA o código con datos de salud ocupacional, aplica estas reglas.

## Regla principal

No copies datos reales identificables en herramientas públicas de IA.

Evita incluir:

- nombres de pacientes, estudiantes, trabajadores o profesionales;
- documentos, correos, teléfonos o direcciones;
- historias clínicas, diagnósticos, incapacidades o restricciones individuales;
- nombres de empresas o instituciones no públicas;
- combinaciones de datos que permitan identificar a una persona.

## Cómo practicar de forma segura

- Usa los datos sintéticos de [[06-recursos/datasets-ejemplo|Datasets de ejemplo]].
- Trabaja con información agregada por área, mes o tipo de evento.
- Cambia nombres por códigos ficticios.
- Elimina columnas innecesarias antes de compartir datos.
- Consulta políticas institucionales antes de subir archivos.

## Decisiones que no deben automatizarse

No uses IA como base única para diagnóstico, triaje, aptitud ocupacional, sanciones, priorización de atención o decisiones individuales. La IA puede ayudar a preparar preguntas, resumir información o crear prototipos, pero la decisión requiere revisión humana autorizada.
