# IA práctica para salud ocupacional

Este vault acompaña una introducción práctica a la programación con inteligencia artificial para estudiantes y profesionales de salud ocupacional.

No necesitas experiencia previa en programación. La idea es aprender a usar herramientas como ChatGPT, Claude y Codex para analizar datos simples, crear prototipos, revisar resultados y comunicar hallazgos de forma responsable.

## Idea central

> La IA no reemplaza el criterio profesional; ayuda a analizar, automatizar, comunicar y prototipar soluciones pequeñas y seguras.

## Audiencia

- Estudiantes de salud ocupacional y seguridad en el trabajo.
- Profesionales de medicina, enfermería y salud ocupacional.
- Personas que quieren empezar a programar con apoyo de IA.

## Advertencia de privacidad

No uses datos reales identificables de pacientes, trabajadores, empresas o instituciones en herramientas públicas de IA. Practica con datos sintéticos, agregados o anonimizados. Valida siempre los resultados con criterio profesional y normativa aplicable.
