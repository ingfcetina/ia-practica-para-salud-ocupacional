# Prompts útiles

tags: #prompts #ia

Un buen prompt incluye contexto, rol, límites y formato esperado.

## Plantilla base

```text
Actúa como [rol].
Contexto: [situación profesional].
Datos disponibles: [qué tengo].
Objetivo: [qué necesito lograr].
Restricciones: no inventes datos, señala dudas, usa lenguaje claro.
Salida esperada: [tabla, lista, resumen, código, checklist].
```

## Para análisis de salud ocupacional

```text
Actúa como analista de salud ocupacional.
Tengo una tabla sintética de ausentismo con área, mes y días de ausencia.
Identifica patrones, preguntas de investigación y posibles acciones preventivas.
No inventes causas. Separa hechos observables de hipótesis.
```

## Para código Python

```text
Ayúdame a escribir código Python para leer un archivo CSV llamado ausentismo_ejemplo.csv,
contar días de ausencia por área y crear un gráfico de barras.
Explícame cada línea para una persona principiante.
```

## Para Codex

```text
Crea una página HTML de un solo archivo para un checklist de seguridad ocupacional.
Debe tener preguntas sí/no, calcular porcentaje de cumplimiento y mostrar recomendaciones.
No uses librerías externas.
```
