# Qué es programar con IA

tags: #conceptos #ia #programacion

Programar con IA significa usar herramientas como ChatGPT, Claude o Codex para transformar una intención en instrucciones, código, análisis o prototipos.

## Método mínimo

1. Describe el problema profesional.
2. Aclara qué datos tienes y qué datos no tienes.
3. Pide una salida pequeña.
4. Prueba con datos sintéticos.
5. Corrige el resultado.
6. Valida antes de usarlo.

## Ejemplo de mejor prompt

```text
Tengo una tabla sintética con área, mes y días de ausencia.
Quiero un gráfico por área, un resumen ejecutivo y preguntas para investigar causas.
No inventes datos y separa hechos de hipótesis.
```
