# Práctica 4: dashboard básico

tags: #practica #dashboard #python

Objetivo: mostrar indicadores simples con tablas y gráficos.

## Indicadores iniciales

Usa `assets/datos/indicadores_dashboard_ejemplo.csv` como tabla sintética para practicar.

- días de ausencia por área;
- incidentes por tipo;
- tendencia por mes;
- porcentaje de cumplimiento de checklist.

También puedes reutilizar los gráficos generados por:

```bash
python scripts/01_analizar_ausentismo.py
python scripts/02_graficar_incidentes.py
```

## Prompt

```text
Ayúdame a diseñar un dashboard básico para una reunión de salud ocupacional.
Solo tengo indicadores agregados por área y mes.
Propón 4 visualizaciones, qué pregunta responde cada una y qué advertencia debo incluir.
```
