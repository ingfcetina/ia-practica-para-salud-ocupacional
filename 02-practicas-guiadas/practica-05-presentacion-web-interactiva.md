# Práctica 5: presentación web interactiva

tags: #practica #web #presentacion

Objetivo: usar una página web simple como material de capacitación o presentación.

## Ejemplo incluido

Abre `assets/checklist_interactivo.html` en un navegador. No requiere servidor ni instalación.

## Prompt para Codex

```text
Crea un archivo HTML único para una actividad de capacitación en seguridad ocupacional.
Debe incluir preguntas, botones, resultado de cumplimiento y recomendaciones.
Usa lenguaje claro para principiantes y no uses librerías externas.
```
