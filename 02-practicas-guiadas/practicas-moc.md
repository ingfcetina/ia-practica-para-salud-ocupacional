# MOC prácticas guiadas

tags: #moc #practicas #principiantes

Las prácticas guiadas son el camino más acompañado del vault. Están escritas para personas que no son especialistas en programación ni inteligencia artificial.

## Cómo usar estas prácticas

No intentes hacer todo de una vez. Para cada práctica sigue este orden:

```text
1. Leer el objetivo
2. Ubicar el archivo que se usará
3. Abrir el archivo sin programar, si es posible
4. Ejecutar el ejemplo o probar la página
5. Observar el resultado
6. Pedir explicación a la IA
7. Escribir límites y preguntas pendientes
```

## Prácticas en orden recomendado

1. [[02-practicas-guiadas/practica-01-analizar-ausentismo|Práctica 1: analizar ausentismo]]
   - Aprendes a leer un CSV, ejecutar un script y revisar un gráfico.

2. [[02-practicas-guiadas/practica-02-clasificar-incidentes|Práctica 2: clasificar incidentes]]
   - Aprendes a organizar descripciones sintéticas y comparar clasificación humana con IA.

3. [[02-practicas-guiadas/practica-03-alertas-riesgo-ocupacional|Práctica 3: alertas de riesgo ocupacional]]
   - Aprendes a crear reglas simples de alerta sin convertirlas en decisiones automáticas.

4. [[02-practicas-guiadas/practica-04-dashboard-basico|Práctica 4: dashboard básico]]
   - Aprendes a elegir indicadores y explicar qué pregunta responde cada gráfico.

5. [[02-practicas-guiadas/practica-05-presentacion-web-interactiva|Práctica 5: presentación web interactiva]]
   - Aprendes a abrir y adaptar una página HTML simple para capacitación.

## Si no puedes ejecutar código

Puedes seguir aprendiendo:

- lee el código;
- mira las imágenes ya generadas en `assets/imagenes/`;
- abre los notebooks en GitHub solo para lectura;
- pide a una IA que explique el código línea por línea;
- vuelve a la instalación después.

## Regla de seguridad

En todas las prácticas:

- usa datos sintéticos;
- no pegues datos reales identificables en IA;
- no hagas conclusiones causales rápidas;
- no conviertas resultados en sanciones o decisiones individuales.
