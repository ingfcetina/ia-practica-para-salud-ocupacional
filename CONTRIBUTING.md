# Contribuir

Este repositorio es un vault educativo. Las contribuciones deben mantener el material claro, seguro y útil para personas sin experiencia previa en programación.

## Principios

- Primero la seguridad y la privacidad.
- Usar datos sintéticos o anonimizados.
- Explicar conceptos antes de mostrar código.
- Preferir ejemplos pequeños y verificables.
- Evitar jerga técnica innecesaria.

## Formato de notas

Cada nota debería incluir, cuando aplique:

- título claro;
- una idea principal;
- ejemplo práctico;
- advertencia de privacidad si se trabaja con datos;
- sección `Relacionado` con wikilinks internos.

Ejemplo:

```markdown
## Relacionado

- [[00-empezar-aqui/mapa-del-vault|Mapa del vault]]
- [[06-recursos/prompts|Banco de prompts]]
```

## No incluir

- Datos reales identificables de pacientes o trabajadores.
- Historias clínicas, incapacidades o diagnósticos individuales.
- Nombres de empresas o instituciones no públicas.
- Código complejo sin explicación.
- Recomendaciones clínicas, ocupacionales o legales definitivas generadas por IA.

## Antes de proponer cambios

- [ ] Revisa [[01-conceptos-clave/limites-riesgos-etica|Límites, riesgos, ética y privacidad]].
- [ ] Confirma que los links internos funcionan.
- [ ] Mantén los ejemplos aptos para una clase introductoria.
