# Página web interactiva para una presentación

tags: #mini-proyecto #html #presentacion #principiantes

Objetivo: mostrar que una página web simple puede servir como material de capacitación o presentación interactiva.

No necesitas crear una aplicación compleja. Puedes empezar con un solo archivo HTML.

## Archivo incluido

Abre este archivo en tu navegador:

```text
assets/checklist_interactivo.html
```

## Qué demuestra

- HTML organiza el contenido.
- CSS mejora la presentación visual.
- JavaScript agrega interacción.
- La IA puede ayudar a construir prototipos pequeños.

## Qué puedes reemplazar

En una página HTML puedes cambiar:

| Parte | Qué cambia |
|---|---|
| `<title>` | Nombre que aparece en la pestaña del navegador. |
| `<h1>` | Título principal visible. |
| Texto de párrafos | Instrucciones para la actividad. |
| Lista de preguntas | Contenido del checklist. |
| Colores CSS | Apariencia visual. |
| Recomendaciones | Mensajes finales para participantes. |

## Paso a paso para principiantes

1. Haz una copia de `assets/checklist_interactivo.html`.
2. Renómbrala, por ejemplo:

```text
assets/presentacion_ergonomia.html
```

3. Abre la copia en un editor de texto.
4. Cambia el título y las preguntas.
5. Guarda.
6. Haz doble clic para abrir en el navegador.
7. Prueba la interacción.
8. Si funciona, úsala en la presentación.

## Prompt para crear una nueva página

```text
Crea una página HTML de un solo archivo para una presentación de 10 minutos.
Tema: uso responsable de IA en salud ocupacional.
Audiencia: profesionales de la salud sin experiencia en programación.
Incluye: título, explicación breve, actividad interactiva, advertencia de privacidad y cierre.
No uses librerías externas.
Agrega comentarios en el código indicando qué partes puedo reemplazar.
```

## Prompt para adaptar la página existente

```text
Adapta assets/checklist_interactivo.html para una presentación sobre [tema].
Mantén un solo archivo HTML.
Indica exactamente qué textos reemplazaste.
Conserva lenguaje claro, no punitivo y orientado a aprendizaje.
```

## Más robusto: incluir gráficos

Puedes incluir imágenes ya generadas:

```text
assets/imagenes/ausentismo_por_area.png
assets/imagenes/incidentes_por_tipo.png
```

Prompt:

```text
Modifica esta página HTML para mostrar una imagen llamada assets/imagenes/ausentismo_por_area.png.
Agrega una pregunta de discusión debajo de la imagen.
No hagas análisis causal.
```

## Cuidado con datos reales

Si vas a mostrar una página en clase o reunión:

- no incluyas nombres;
- no incluyas datos clínicos;
- no incluyas identificadores de trabajadores;
- no incluyas nombres de instituciones si no corresponde;
- usa datos sintéticos para practicar.
