# Página web interactiva para una presentación

tags: #mini-proyecto #html #presentacion

Objetivo: mostrar que una página web simple puede ser una herramienta de capacitación.

## Archivo incluido

Abre este archivo en tu navegador:

```text
assets/checklist_interactivo.html
```

## Qué demuestra

- HTML organiza el contenido.
- CSS mejora la presentación.
- JavaScript calcula resultados.
- La IA puede ayudar a construir prototipos pequeños.

## Actividad práctica

1. Abre el archivo.
2. Marca respuestas del checklist.
3. Observa cómo cambia el porcentaje.
4. Pide a Codex una adaptación para otro tema.
