# Checklist inteligente

tags: #mini-proyecto #checklist #web

Objetivo: crear una actividad interactiva para inspección o capacitación.

## Qué hace

El ejemplo `assets/checklist_interactivo.html` permite marcar respuestas sí/no, calcular cumplimiento y mostrar recomendaciones.

También tienes una tabla sintética para practicar análisis de checklist en `assets/datos/checklist_inspeccion_ejemplo.csv`.

## Adaptaciones posibles

- elementos de protección personal;
- orden y aseo;
- seguridad en laboratorio;
- ergonomía básica;
- preparación ante emergencias.

## Prompt para modificarlo

```text
Adapta este checklist para una capacitación de ergonomía en oficina.
Mantén un solo archivo HTML, lenguaje claro y recomendaciones no punitivas.
```
