# MOC mini proyectos

tags: #moc #mini-proyectos

Los mini proyectos son actividades un poco más abiertas que las prácticas guiadas. Sirven para aplicar lo aprendido y empezar a adaptar ejemplos.

## Cómo trabajar un mini proyecto

```text
1. Leer el objetivo
2. Revisar datos o archivo base
3. Hacer una copia antes de modificar
4. Pedir a la IA un plan
5. Cambiar una cosa pequeña
6. Probar
7. Documentar límites y privacidad
```

## Regla para principiantes

No intentes cambiar todo al mismo tiempo. Si algo funciona, cambia solo una parte:

- una pregunta;
- un título;
- una ruta de archivo;
- una columna;
- un color;
- una sección del informe.

## Prompts útiles

```text
Quiero adaptar este mini proyecto.
Primero explícame qué archivo debo abrir, qué parte puedo reemplazar y cómo probar si funcionó.
No cambies todo el proyecto de una vez.
```

```text
Revisa este cambio como si fueras docente de una persona principiante.
Dime si el resultado sigue siendo claro, seguro y sin datos identificables.
```
