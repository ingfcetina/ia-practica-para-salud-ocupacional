# Codex

tags: #codex #codigo #herramientas

Codex sirve para trabajar dentro de un proyecto: crear archivos, modificar código, ejecutar pruebas y explicar cambios.

## Buen uso en este vault

- Crear ejemplos HTML de una sola página.
- Generar scripts Python simples.
- Revisar enlaces Markdown.
- Mantener estructura de carpetas.

## Prompt seguro

```text
Crea un ejemplo didáctico y mínimo.
No uses datos reales.
Explica qué archivos vas a crear.
Mantén el código simple para principiantes.
```

## Diferencia con ChatGPT web

ChatGPT puede ayudarte a pensar. Codex puede ayudarte a editar el repositorio y validar archivos.
