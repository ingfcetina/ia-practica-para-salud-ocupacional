# Claude

tags: #claude #herramientas

Claude es útil para trabajar con textos largos: protocolos, guiones, documentos de capacitación o borradores de políticas.

## Buen uso

```text
Revisa este guion de capacitación sobre riesgos ocupacionales.
Hazlo más claro para una audiencia no técnica.
Devuelve una versión de 10 minutos, una actividad práctica y 5 preguntas de cierre.
```

## Cuándo elegirlo

- Cuando el documento es extenso.
- Cuando quieres mejorar estructura y tono.
- Cuando necesitas comparar versiones de una explicación.
