# ChatGPT web

tags: #chatgpt #herramientas

ChatGPT web es útil para explicar conceptos, crear borradores, generar prompts y analizar datos pequeños pegados como texto.

## Buen uso

- Pedir explicación de código línea por línea.
- Convertir una tabla agregada en resumen ejecutivo.
- Diseñar preguntas para investigar causas.
- Preparar actividades de capacitación.

## Prompt inicial

```text
Actúa como tutor de programación para estudiantes y profesionales de salud ocupacional.
Explícame este código de Python en lenguaje simple, sin asumir experiencia técnica.
Incluye una analogía con una tabla de Excel.
```

## Cuidado

No pegues datos identificables. Usa datos sintéticos o agregados.
