| Herramienta | Enlace oficial | Para qué se usa |

| Python | <https://www.python.org/downloads/> | Ejecutar scripts y notebooks. |

| pip / paquetes de Python | <https://packaging.python.org/en/latest/tutorials/installing-packages/> | Instalar librerías como pandas y matplotlib. |
| Git | <https://git-scm.com/downloads> | Clonar repositorios y trabajar con control de versiones. |
| Libro oficial de Git | <https://git-scm.com/book/en/v2> | Aprender conceptos de Git. |
| GitHub Docs | <https://docs.github.com/> | Usar repositorios, clonar, descargar y colaborar. |
| VS Code | <https://code.visualstudio.com/Download> | Editor recomendado para scripts y notebooks. |
| Extensión Python para VS Code | <https://marketplace.visualstudio.com/items?itemName=ms-python.python> | Ejecutar Python en VS Code. |
