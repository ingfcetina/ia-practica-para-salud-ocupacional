# Banco de prompts

tags: #prompts #recursos

## Analizar datos agregados

```text
Actúa como analista de salud ocupacional.
Analiza estos datos agregados. Separa hechos observables, hipótesis y preguntas pendientes.
No inventes causas. No hagas recomendaciones individualizadas.
```

## Explicar código

```text
Explica este código como si yo supiera usar Excel pero estuviera empezando con Python.
Define cada palabra técnica y dame una versión resumida.
```

## Crear actividad

```text
Diseña una actividad de 10 minutos para profesionales de salud ocupacional.
Debe usar un ejemplo cotidiano, una pregunta de discusión y una advertencia ética.
```

## Revisar privacidad

```text
Revisa este caso de ejemplo y dime si contiene datos que podrían identificar a una persona o institución.
Sugiere cómo anonimizarlo antes de usar una herramienta de IA pública.
```
